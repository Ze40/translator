# Транслятор C# $\rightarrow$ PHP

## Входной алфавит лексем (C#)

### Идентификаторы **I**:

- Имена переменных
- Имена функций
- Имена классов

### Ключевые слова **W**:

- using
- namespace
- class
- static
- void
- int
- string
- float
- double
- bool
- if
- else
- for
- while
- return
- new
- true
- false

### Числовые константы **N**:

- Целые: 123, 0.
- Вещественные (с фикс. точкой): 3.14, .5, 10.
- С плавающей точкой: 1e-10, 2.5E+3.

### Строковые и символьные константы **C**:

- Строки: "hello world".
- Символы: 'a', '\n'.

### Операции **O**:

- Арифметические: +, -, \*, /, %.
- Сравнения: ==, !=, <, >, <=, >=.
- Логические: &&, ||, !.
- Присваивания: =, +=, -= и т.д. (можно начать с =).
- Разделители: ; , . ( ) { } [ ].

### Комментарии:

- Строчные: // ...
- Блочные: /_ ... _/

### Пропуски:

- Пробелы
- Табуляции
- Новые строки

> Должны игнорироваться сканером, но могут быть важны для подсчета строк.
